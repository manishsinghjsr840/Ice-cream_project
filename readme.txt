# Ice Cream Selling Website

Description
This project is an Ice Cream Selling Website built using the Django framework. The website allows users to browse various ice cream products, learn about the company, and contact the business through a contact form. The contact form submissions are saved in the admin database, allowing the business to manage and respond to customer inquiries efficiently.


Features
Home Page: Displays an overview of the ice cream offerings and company information.
About Us Page: Provides detailed information about the company, its history, and mission.
Services Page: Lists the different types of ice creams and services offered by the company.
Contact Page: Allows users to fill out a contact form with their name, email, phone number, and message. The submissions are saved in the admin database using a custom model.


Technologies Used
Django
HTML
CSS
Bootstrap
Python


Usage
Open your web browser and navigate to http://127.0.0.1:8000/ to view the homepage.
Use the navigation bar to explore the different sections of the website.
Fill out and submit the contact form on the "Contact Us" page to send a message to the business.
Log in to the admin panel at http://127.0.0.1:8000/admin/ using the superuser credentials to view and manage contact form submission
